# Ruff backlog (post-audit)

Короткий технический долг по линтеру, который не блокирует текущий cleanup-эпик.

## Текущее состояние

- На полном `python -m ruff check src/` есть исторические замечания вне изменённых файлов.
- Наиболее частые категории:
  - `I001` — неотсортированные import block.
  - `S112` — `try/except/continue` без явного логирования контекста.
  - Точечные `F*`/`UP*` на отдельных модулях.

## Пакетный план исправлений

1. **Imports pass**: пакетно прогнать `ruff --fix` по модулям `admin/*`, затем `bot/*`.
2. **Exception pass**: для hot-path файлов (`bot/main.py`, `bot/journal_handlers.py`, `bot/sender.py`, `matrix_send.py`) заменить немые broad-except на typed-except или лог с контекстом.
3. **Typing hygiene**: после import/exception pass закрепить чистый `ruff check src/` в CI как обязательный без локальных обходов.

## Вне скоупа

- Массовое изменение бизнес-логики.
- Тотальная переработка всех блоков обработки ошибок в одном PR.
